Heritage Furniture is a family-owned business located in Nottingham, United Kingdom. They have 12 locations throughout the UK and have been online since 2007.

The Project

The current Heritage Ecommerce platform ran on OS Commerce. I migrated the existing sales orders and customer data to our new Shopify platform. I designed a new interface using the updating branding supplied by Heritage's in-house design team.

I also designed and built a custom React Native application that used Shopify's API to handle online sales and integrated with Heritage's new Ecommerce website to allow customers to place new orders and view previous orders through the new app.
